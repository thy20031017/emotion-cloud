# 情绪云 🌥️

一个温暖的心理健康可视化应用，帮助你记录和感知每日情绪变化。

## 功能特点

- **情绪云交互**：点击四种情绪云（焦虑、低落、开心、平静），触发独特的治愈动画
- **日历记录**：按日期查看情绪状态，颜色直观呈现
- **数据面板**：实时监测压力值、情绪效价、唤醒度
- **情绪轨迹**：可视化展示全天候情绪波动
